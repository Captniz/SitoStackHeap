What is this website?


This website was created to explain in an easy and clear way the inner workings of the garbage collector and memory management in Java. We'll go trough the concepts of STACK e HEAP and how they are used to manage the memory of the program.

This website was created by Simone Tardiani using SCSS, HTML and Javascript as a class project. It's written entirely in english so it can be understood by anyone and it's developed keeping so it can eventually be updated and mantained.
